Privacy Policy for ZodiacAI Finder

Last Updated: October 16, 2025

App Name: ZodiacAI Finder - Astrology & Horoscope

Developer: kregg

Data Collection

ZodiacAI Finder collects the following types of user data:

  • Personal Information: Optional name, birth date, birth time, and birth location (provided by user for astrology calculations)
  • User-Generated Content: Chat messages with AI, palm reading images
  • Device Information: Device details and app usage statistics (automatically sent with API requests)

Data Usage

Collected data is used solely for:

  • Providing personalized astrology readings, horoscopes, and AI-powered features
  • App functionality and improvement

Data Sharing

User data is shared with Pollination AI (text.pollinations.ai) for AI processing. Data is encrypted in transit (HTTPS) and not sold to third parties. Some responses may appear in public feeds unless marked as private.

Data Security

We implement appropriate security measures to protect user data. Data is not stored locally on devices or our servers beyond real-time processing.

Data Deletion Request

How to Request Data Deletion:

  1. Contact us at kreg9da@gmail.com with the subject "Data Deletion Request - ZodiacAI Finder"
  2. Include your name (if provided) and any relevant details about your data
  3. We will process your request within 30 days

What Happens When You Request Deletion:

  • Since ZodiacAI Finder does not store user data locally or on servers, no data is retained after real-time processing
  • Data sent to Pollination AI for processing is handled according to their privacy practices
  • We will confirm deletion and cease any further processing of your information
  • No additional retention period applies - data is processed and immediately discarded
